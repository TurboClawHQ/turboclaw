## What is TurboClaw?

TurboClaw takes the powerful OpenClaw engine and wraps it in a **zero-friction installation experience**:

- **One-command install** - `npm install -g turboclaw && turboclaw init`
- **Pre-configured LLM proxy** - No API key juggling. Works out of the box.
- **Personality wizard** - Create your agent's unique personality in 8 steps
- **Multi-channel ready** - WhatsApp, Discord, Slack, Telegram, phone calls
- **Native macOS app** - Beautiful dashboard, no terminal needed
- **European DNA** - GDPR compliant, Polish & EU language support

## Quick Start

```bash
npm install -g turboclaw
turboclaw init
```

That's it. Your agent is live.
